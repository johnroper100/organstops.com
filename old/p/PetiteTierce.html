Petite Tierce French

Mentioned only by Douglass, who writes:

Some builders used [the name Grosse Tierce] for the Tierce 1-3/5', in order to differentiate it from the narrow scale seventeenth, as occasionally found in organs earlier in the century (Petite Tierce).

Examples

Petite Tierce 1-3/5', Grand Orgue; Basilica, St. Hubert, Belgium; le Picard 1685. This is the only known example of this name. The same division contains a Grosse Tierce at 3-1/5' pitch.
